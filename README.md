# 🌐 ESP32 IoT Temperature & Humidity Monitor

## Overview
This project is an IoT-based environmental monitoring system using an ESP32 and DHT11 sensor. It measures real-time temperature and humidity and displays the data on a web dashboard accessible from any device on the same network.

---

## Features:
- WiFi-enabled ESP32 web server
- Real-time temperature & humidity monitoring
- Live browser dashboard
- Accessible via phone or laptop
- Auto-refreshing interface

---

## Hardware Components
- ESP32 Development Board
- DHT11 Sensor
- Breadboard & Jumper Wires

## How It Works:
1. ESP32 connects to WiFi
2. Reads data from DHT11 sensor
3. Hosts a web server
4. Displays readings on a web page

---

## 🌍 Accessing the Dashboard:

1. Upload the code
2. Open Serial Monitor
3. Copy the IP address
4. Open in browser:


http://<ESP32-IP>
